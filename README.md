Readme 
MainActivity: The entry point. Handles the "Start Quiz" button.

QuizActivity: Manages the quiz logic, iterates through the question list, and handles button clicks for "Hack" (True) or "Myth" (False).

ScoreActivity: Displays the final score and uses a when expression to determine the user's rank.
activity_main.xml: Contains btnStart.

activity_main2.xml: Contains txtQuiz, btnHack, btnMyth, and btnNext.

activity_main3.xml: Contains txtTotal, txtFeedback, and btnReview.
